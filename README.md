## 🌟 简介

一个免费的PDF文献翻译工具，让看英文文献像喝水一样简单。


## 📦 使用方式

- windows 用户:

  直接下载安装包：https://pan.baidu.com/s/1vhrZLiiwBQuWJ4PZ0bJQ_w?pwd=8888 (提取码: 8888)

- 其它系统用户:

  未提供安装包，可安装相关环境，用 python 运行 `main.py`。

翻译完的PDF文件，会在其对应目录下生成 `-dual.pdf`(双语对照文件) 和 `test-mono.pdf`(中文翻译文件)
